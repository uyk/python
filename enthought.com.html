Python for Excel

Free 30-day Trial

PyXLL is an Excel plug-in that allows organizations to seamlessly deploy Python algorithms and models to all their Excel users.

PyXLL has allowed investment banks to implement sophisticated financial models, online retailers to create interactive graphical analysis tools to segment customer data, and business analysts to update data models with real-time web data feeds.

All driven by Python, delivered via Excel.

Try PyXLL free for 30 days to see how it can transform your organization's use of Excel.
